关于Object

2021-01-03
java基础-日学记我不是咸鱼

是什么

Object是一个类。位于java. Lang包中，它是java所有类的父类，何一个类时候如果没有明确的继承一个父类的话，那么它就是Object的子类；这就意为着一下两个类的定义是完全一样的

有什么用

Object作为所有类的父类，我认为主要体现了两个功能

  1. 接收所有类的对象
  2. 为所有类提供一些基础方法，提高代码的复用性

hashCode()

hashCode() 返回哈希值，而 equals() 是用来判断两个对象是否等价。等价的两个对象散列值一定相同，但是散列值相同的两个对象不一定等价，这是因为计算哈希值具有随机性，两个值不同的对象可能计算出相同的哈希值。

3. 深拷贝

4. clone() 的替代方案

使用 clone() 方法来拷贝一个对象即复杂又有风险，它会抛出异常，并且还需要类型转换。Effective Java 书上讲到，最好不要去使用 clone()，可以使用拷贝构造函数或者拷贝工厂来拷贝一个对象。
