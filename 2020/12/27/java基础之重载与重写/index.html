java基础之重载与重写

2020-12-27

是什么

重写

子类对父类方法的重新实现，两个方法返回值、形参必须一致，主要为实现内容可以不同。

重载

同一类中，同名方法，以参数、返回类型的不同进行区分（参数类型列表必须不同），针对不同参数采用不同的处理方式。

为什么

java多态性的一种体现，便于方法整合，重写可以针对同一种方法，不同的对象有不同的具体实现方式。比较常见的例如object的tostring与equal方法。重载可以依据输入参数不同调用不同方法。

怎么做

重写equals方法

重写obj的equals方法以记录用法（一般来说要一起重写hashcode）

@Override
public boolean equals(Object obj) {
    if(this == obj){
        return true;//地址相等
    }

    if(obj == null){
        return false;//非空性：对于任意非空引用x，x.equals(null)应该返回false。
    }

    if(obj instanceof user){
        user other = (user) obj;
        //需要比较的字段相等，则这两个对象相等
        if(equalsStr(this.name, other.name)
                && equalsStr(this.age, other.age)){
            return true;
        }
    }

    return false;
}
