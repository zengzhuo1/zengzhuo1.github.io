java容器基础

java容器-日学记我不是咸鱼

是什么

       顾名思义容器就是装东西的器具，具体到java中容器其实是一种数据结构用于储存数据，也可以理解成能够帮助你管理对象间的关系的一个java程序，通常容器都是继承自Collection(集合)或Map(图)常用的容器如下图所示

为什么

这是因为常规的存储结构（数组）在不同应用场景下存在不同的缺陷（例如插入数据需要全体移位），这个时候为了不同的应用场景设计采用不同的数据存储结构利于提高程序效率，当然你不可能每次都自己写吧（要你设计书写你也不一定能弄得出来啊），所以java提供了各种不同存储结构的容器类来帮助你管理数据。可见在不同应用场景选择适合的容器来存储数据很有必要。
